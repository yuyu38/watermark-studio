## ✨ 功能特性

### 核心功能
- 📷 **图片获取**：支持从相册选择
- 👁️ **实时预览**：所有参数调整即时反映
- 💾 **导出功能**：保存到相册或复制到剪贴板
- 🔒 **隐私保护**：100%本地处理，无需联网

### 水印编辑
- 📝 自定义水印文字（默认："该证件仅供XX查看，不得他用"）
- 📏 字体大小调整（12-200px）
- 🎚️ 透明度调整（10%-100%）
- 🎨 文字颜色选择
- 📋 自动换行（5字/行）

### 布局模式

**单个水印模式**
- 🎯 9宫格位置选择（9个预设位置）
- 📐 自适应边距，保护图片边缘

**平铺水印模式（默认）**
- 🔲 整张图片均匀平铺
- ↔️ 可调平铺间隔（30-600px）
- 🔄 可调旋转角度（-60°~+60°）

```

├── index.html              # 主页面
├── css/style.css          # 样式文件
└── js/
    ├── main.js            # 应用主逻辑
    ├── watermark-engine.js # 水印渲染引擎
```
